# S&P 500 Stock Analysis (2014-2017)

<h2 align="center">Overview</h2>

This project analyzes historical stock price data of S&P 500 companies from 2014 to 2017. The goal is to understand overall market trends, stock performance, volatility patterns, and year-wise growth using Python-based data analytics techniques.

---

<h2 align="center">Data Source</h2>

The dataset was obtained from [Maven Analytics](https://www.mavenanalytics.io/), which provides high-quality financial datasets for analysis. It contains daily stock price records including stock symbol, date, open, high, low, close, and volume for S&P 500 listed companies.

---

<h2 align="center">Objective</h2>

- Analyze overall market behavior and stock performance.
- Identify high-performing and volatile stocks.
- Explore year-wise trends and trading volumes.
- Apply EDA and visualizations for insights supporting basic investment decisions.

---

<h2 align="center">Process</h2>

1. **Data Loading & Initial Overview**: Import dataset, inspect records, data types, missing values, and summary statistics.  
2. **Data Cleaning & Preprocessing**: Handle missing values, remove duplicates, convert date types, and create features like daily returns and moving averages.  
3. **Exploratory Data Analysis (EDA) & Visualization**: Visualize market trends, top traded stocks, volatility patterns, and moving averages using line charts, bar charts, histograms, and correlation heatmaps.  
4. **Insights & Findings**: Identify trends, high-performing stocks, sector dominance, and risk patterns.

---
